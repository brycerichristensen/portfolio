Propulsion Project

I’ve been leading the controls team for BYU Propulsion Research since Spring 2025, after joining the team in Fall 2024. My focus has been on improving the reliability and robustness of our static fire control system and designing the software and hardware for our first flight configuration control system, planned to launch in Spring 2026.

Static Configuration

This project focused on optimizing and refining the static configuration’s data collection and control system. Key objectives included reducing signal noise in the sensors, improving reliability and robustness, and cutting down setup time to streamline launch-day operations, as electronics setup had previously been the primary bottleneck.
